# Security and Responsible Use

LabMentor is a training companion for authorized pentest labs and learning environments.

## Responsible use

Use LabMentor only where you have permission, such as:

- your own lab
- private vulnerable VMs
- authorized training platforms
- classroom or workshop environments
- formal pentest labs where the rules permit this type of tooling

Do not use LabMentor for:

- unauthorized targets
- real third-party systems without permission
- certification exams where AI/LLM/tool assistance is prohibited
- bypassing platform rules
- automated exploitation or flag submission

## Reporting security issues

If you find a security issue in LabMentor itself, please open a GitHub issue with enough detail to reproduce the problem. Do not include sensitive information from private labs or real targets.

## Design boundaries

LabMentor intentionally focuses on:

- methodology
- notes
- next-step guidance
- walkthrough learning
- safe lab workflows

LabMentor intentionally avoids:

- autopwn behavior
- blind exploitation
- destructive testing
- public target automation
- exam assistance
